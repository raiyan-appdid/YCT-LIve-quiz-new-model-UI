Audio files in this folder are generated locally (original, no external download).
The app expects these exact names:

- lobby_loop.wav    (waiting room background loop)
- countdown_tick.wav
- game_start.wav
- answer_select.wav
- answer_correct.wav
- answer_wrong.wav
- timer_tick.wav
- time_up.wav
- transition.wav
- game_end.wav

Audio is optional at runtime. If a file is missing, the app will continue silently.
